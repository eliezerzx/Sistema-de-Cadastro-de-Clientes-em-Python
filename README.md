Sistema de Cadastro de Clientes em Python

📌 Descrição

Mini-sistema em Python para cadastro de clientes com menu interativo, busca por email e exportação para CSV.
Ideal para praticar lógica de programação, funções e manipulação de arquivos.

🚀 Funcionalidades
Adicionar clientes (nome, idade, email)

Listar clientes cadastrados

Buscar clientes pelo email

Exportar dados para arquivo CSV

🛠️ Tecnologias utilizadas
Python 3

Biblioteca padrão csv
